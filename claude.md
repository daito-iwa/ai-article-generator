# TechNote - AI記事自動生成プラットフォーム

## プロジェクト概要
Note風の記事投稿プラットフォームで、ユーザーが記事を投稿でき、AIが自動でトレンド記事を生成する完全無料の収益化システム。GitHub Pages上で動作し、完全な匿名性を保ちながら24時間365日自動運営。

## 核心価値
- **完全無料**：GitHub Pages + Ollama でAPIコスト0円
- **完全自動**：人の介入なしで記事が増え続ける
- **完全匿名**：すべてイニシャルでプライバシー保護  
- **即座に収益化**：AdSense + ユーザー投稿で自動収益化
- **自然な見た目**：普通の記事投稿サイトに見える
